Key Features

Containerized Application Support

Develop and deploy applications packaged as Docker containers, ensuring consistency across environments.

Streamlit & Next.js Ready

Out-of-the-box support for popular frameworks like Streamlit for data apps and Next.js for full-stack web applications.

Integrated Version Control & Releases

Automated Git-based version control for your code, with the ability to create containerized releases from specific commits, configured via a simple `workbench.yml` file.

Flexible Deployment Options

Releases can be one-shot Python scripts, long-running services, or services with HTTP ports proxied to users.

Identity & Access Management

Leverages Keycloak for robust identity management. Define granular permissions for users, controlling access to applications and resources.

Layered Network Access Control

Two layers of network control: container-to-container communication rules via `workbench.yml` and HTTP(s) access controlled by Traefik (public or authenticated access for static content and services).

One-Click Next.js Deployment

Effortlessly deploy your Next.js applications with a single click.

Integrated Shell & Package Management

Provides an integrated shell and Python package management for each container, streamlining development workflows.

Cloud & Edge Compatible

Designed to run effectively in cloud environments and on edge devices, even without persistent internet connectivity.

What is FloWide Workbench?

Workbench is an integrated development and deployment platform for containerized applications. It simplifies the creation, versioning, and deployment of your projects, offering out-of-the-box support for Streamlit and Next.js based full-stack applications.

The platform is architected from multiple services running in Docker containers, ensuring modularity and scalability.
